Golap Saroar Portfolio This repository contains the source code for the personal portfolio of Golap Saroar, a Computer Science & Engineering undergraduate at RUET. The site is designed with a "Problem-First" engineering philosophy, emphasizing logic and clean architecture over flashy bloat.

🚀 Live Demo View Portfolio Live:https://golapsaroar.github.io/my-portfolio2/

🛠️ Tech Stack & Philosophy This portfolio is a testament to "New-Age Engineering"—leveraging modern browser APIs and clean code to create a premium user experience without heavy frameworks.

Frontend: Semantic HTML5, CSS3 (Modern Flexbox/Grid), and Vanilla JavaScript (ES6+).

Performance: Zero external library dependencies (no jQuery, no React, no Bootstrap).

Typography: Optimized loading of Bebas Neue, Syne, and DM Mono via Google Fonts.

Animations: Powered by the Intersection Observer API for scroll-reveal effects and requestAnimationFrame for a smooth custom cursor.

✨ Key Features

Custom Interactive Cursor A dual-element cursor system featuring a snap-to-dot and a lerp-interpolated ring that lags slightly behind the mouse for a liquid, tactile feel. The ring expands dynamically when hovering over interactive elements like cards and buttons.

Adaptive Typewriter Engine A custom JavaScript typing engine that cycles through core professional identities: Problem Solver, Systems Thinker, Tool Builder, and AI-Augmented Developer.

Smart Scroll Reveal Sections are orchestrated to "rise up" and fade in as the user scrolls, utilizing staggered CSS transition delays to create a sense of rhythm and hierarchy.

Interactive Skill Proficiency Skill bars remain at 0% width until they enter the viewport, at which point they animate to their specific proficiency levels using the Intersection Observer.

Mobile-First Navigation A blurred backdrop (Glassmorphism) navigation bar that transitions into a full-screen, accessible overlay menu on mobile devices.

📂 Project Showcase The portfolio highlights two flagship tools:

Smart Tab Manager: A productivity tool engineered for browser tab management, utilizing the Local Storage API for persistent data.

OOP — The Complete Visual Guide: An interactive reference tool that transforms dense OOP theory into a visual learning experience with keyboard-driven search.
